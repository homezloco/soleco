All notable changes to the Soleco CLI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-02-27

### Added
- Initial release of the Soleco CLI
- Core CLI structure with Click framework
- Configuration management system
- API client for interacting with Soleco backend
- Network commands for checking Solana network status
- RPC commands for listing and analyzing RPC nodes
- Mint commands for tracking and analyzing token mints
- Diagnostics commands for system information
- Interactive shell mode
- Multiple output formats (JSON, CSV, table)
- Comprehensive documentation and examples
- Support for pump token tracking and analysis
- Support for all mint addresses tracking (not just new ones)
- RPC node health check and performance analysis
